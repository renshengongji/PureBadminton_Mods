### Manual install
1. Find the game folder: `steamapps\common\Pure Badminton Demo\Pure Badminton_Data`
2. Make a dir `mods` in the `Pure Badminton_Data`
3. Copy the `characters.json` file by generated to the `mods` folder
4. Launch game

### Auto install
**Windows:** Download and run the `auto.ps1`
**MacOS/Linux:** Download and run `auto.sh`
